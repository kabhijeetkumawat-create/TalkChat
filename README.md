💬 Talk Chat App(Working)

A modern real-time chat application built using Kotlin, MVVM architecture, and Jetpack Compose, powered by Firebase for backend services.

🚀 Features
🔐 User Authentication (Firebase Auth)
💬 Real-time Messaging
👤 One-to-One Chat
⚡ Modern UI with Jetpack Compose
🔄 Live Data Updates using MVVM
☁️ Firebase Integration (Auth + Database)

🛠️ Tech Stack
Language: Kotlin
UI Toolkit: Jetpack Compose
Architecture: MVVM
Backend: Firebase
Firebase Authentication
Firebase Realtime Database / Firestore
Dependency Injection: Hilt
